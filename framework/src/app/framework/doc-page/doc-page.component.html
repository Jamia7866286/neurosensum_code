Dashboard Page Component
Dashboard page is a structural component it wraps the content of a dashboard page within. page is divided into 3 sections:
  • Head: inside a page tag '.page-head' class is used to define heading of a page. user can put any html in head.
  • Body: inside a page tag '.page-content' class is used to define body of a page. user can put any html in body.
  • Footer: inside a page tag '.page-footer' class is used to define footer of a page. user can put any html in footer.
