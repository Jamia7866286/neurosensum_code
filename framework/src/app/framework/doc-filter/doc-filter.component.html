Inputs to be taken

Filter Config: (FilterConfig) It will take following properties:

Question guid(questionGuid) Question guid of the question which is to be binded with filter
Variable id(variableId) variable id of the question which is to be binded with filter
Type(type) type of the filter you want to make
Filter types:
  • Dropdown
  • List
  • Search box
  • Number box
  • Date picker
  • Date range picker
Filter callbacks (optional) Functions which are to be executed at some specific events
Filter callback events:
  • beforeOptionsBind
  • afterOptionsBind
  • onSelectionChanges
Dropdown config Configuration of dropdown (if dropdown filter type is used)
Dropdown Settings:
  • dropdownType: Type of dropdown to make
    Types of dropdown
    • Single Select
    • Multi Select
  • idField: field to be used for value
  • textfield: field to be used for label
  • enableCheckAll: Display checkall button or not
  • selectAllText: Text to display when nothing is selected
  • unSelectAllText: Text to display when all options are selected
  • allowSearchFilter: Display search filter
  • clearSearchFilter: Display option to clear search filter
  • maxHeight: max height to dropdown
  • itemsShowLimit: max items to be displayed which are selected
  • limitSelection: Limit the max selection (-1 is infinite)
  • searchPlaceholderText: Placeholder for search box
  • noDataAvailablePlaceholderText: Placeholder when no data is present
  • closeDropDownOnSelection: Whether to close dropdown on selection
  • showSelectedItemsAtTop: whether to display selected items on top
  • defaultOpen: Keep dropdown open initially
