Dashboard Authentication
Current framework support 2 types of Dashbaords.
  • Anonymous Dashboard: authenticated using auth token.
  • Shared Dashboard: authenticate using username and password.
Anonymous Dashboard Authentication
following are the steps to configure dashbaord for anonymous use: when
  • Step 1 : Pass anonymousToken in framework config at time of import framework in app module.
  • Step 2: In App Component Authenticate dashbaord using auth service and load peoject.

Shared Dashboard Authentication
Shared Dashboard must be authorized using Login component. In which user have to provide username and password which will get validated at surveysensum server.
In framework you can either use login component provided by framework like below code snippet : 
                
                    {{loginHtml}}
