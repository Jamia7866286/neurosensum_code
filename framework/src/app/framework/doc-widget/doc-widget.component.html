Dashboard Widget Component
Dashboard widget is a structural component it wraps the content of a widget within. it should always used inside a Dashboard Page Component body. widget is divided into 3 sections:
  • Head: inside a widget tag '.widget-head' class is used to define heading of a widget. user can put any html in head.
  • Body: inside a widget tag '.widget-content' class is used to define body of a widget. user can put any html in body.
  • Footer: inside a widget tag '.widget-footer' class is used to define footer of a widget. user can put any html in footer.
