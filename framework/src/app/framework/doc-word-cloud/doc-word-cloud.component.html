Word Cloud
Inputs to be taken

textVariableId: (String) It will take variable Id of the words which are to be set in word cloud.
strengthVariableId: (String) It will take the strength variable Id of that given word which will determine its size in the wordcloud.

Optional Inputs

beforeGettingResponseData: (Function) There will be 3 parameters that will be passed i.e- tableDefination, pageIndex and pageSize to alter Echart options before getting response data.
beforeCloudRender: (Function) There will be 3 parameters that will be passed i.e- tableDefination, pageIndex and pageSize to alter Echart options before rendering Word cloud.
